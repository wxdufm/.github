# where am i?

hey! welcome to the github organization of wxdu 88.7 fm, located in durham nc. we are the non-commercial volunteer-run radio station of duke university, and a service of the duke university union. this is the public repository for our technology operations. email us if you have questions :)

# how can i help?
if you'd like to contribute to our open-source software, follow these steps:
  - clone the relevant repo locally
  - make a new branch and code in that branch
  - once you're ready, make a pull request to the wxdufm repo
  - our computing team will look over your code and approve it
